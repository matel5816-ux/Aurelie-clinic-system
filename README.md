# 【醫美診所】預約排程管理系統

一個 Notion 風格的醫美預約管理系統，提供表格、日曆和看板多視圖，以及自動化的預約提醒模板功能。

## 功能特性

### 📊 表格檢視
- 查看所有預約信息
- 快速編輯和刪除預約
- 支持實時搜索和過濾

### 📅 日曆檢視
- 月曆視圖
- 預約信息一目了然
- 支持月份切換

### 📌 看板檢視
- 多種分組方式：療程項目、預約日期、負責人員
- 卡片式展示預約信息
- 直觀的任務管理

### 📝 預約提醒模板
- 預設 LINE 提醒文字模板
- 一鍵複製，快速發送提醒
- 支持自訂模板

## 預約欄位

1. **客人姓名** - 預約客人的名字
2. **預約日期時間** - 預約的具體日期和時間
3. **療程項目** - 選擇項目：雷射、微整、清粉刺、其他
4. **負責人員** - 負責該預約的員工名字
5. **電話號碼** - 客人的聯絡電話
6. **通知狀態** - 標記是否已通知客人（未通知、已通知）

## 開始使用

### 1. 安裝 Node.js

如果還沒有安裝 Node.js，請到 [nodejs.org](https://nodejs.org/) 下載並安裝最新 LTS 版本。

### 2. 安裝依賴

在項目根目錄打開終端，執行：

```bash
npm install
```

### 3. 啟動開發伺服器

```bash
npm run dev
```

然後在瀏覽器中打開 `http://localhost:5173` 即可使用。

### 4. 構建生產版本

```bash
npm run build
```

## 數據存儲

所有數據都存儲在瀏覽器的 `localStorage` 中，刷新頁面後數據仍會保留。

> 💡 提示：清除瀏覽器 Cookie 會刪除所有數據，請定期備份重要信息。

## 使用建議

### 快速新增預約

1. 點擊任何檢視下的「+ 新增預約」按鈕
2. 填寫所有必填欄位
3. 點擊「新增」保存

### 使用提醒模板

1. 切換到「📝 提醒模板」標籤
2. 點擊預設模板上的「複製」按鈕
3. 將複製的文本粘貼到 LINE 或其他聊天應用
4. 根據需要修改預約時間和項目信息

### 切換視圖

- 頂部導航欄可快速切換表格、日曆、看板檢視
- 在看板檢視中，可以選擇不同的分組方式

## 常見問題

### Q: 我的數據會丟失嗎？
A: 數據存儲在瀏覽器的 localStorage 中，清除瀏覽器數據時會被刪除。建議定期備份重要信息。

### Q: 可以添加更多療程項目嗎？
A: 可以的。編輯 `src/types.ts` 中的 `TherapyType` 類型定義，然後在 `TableView.tsx` 和其他組件中更新相應的列表。

### Q: 如何備份我的數據？
A: 在瀏覽器開發者工具 (F12) 的 Application 標籤中，找到 localStorage，複製相關的值進行備份。

## 開發指南

### 添加新欄位

1. 在 `src/types.ts` 中的 `Appointment` 介面添加新欄位
2. 在 `AppointmentForm.tsx` 中添加對應的表單欄位
3. 在各個檢視組件中更新顯示邏輯

### 自訂樣式

所有樣式文件位於 `src/styles/` 目錄，使用 CSS 變量進行顏色和間距管理，方便統一修改。
